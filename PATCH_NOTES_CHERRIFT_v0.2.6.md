# CHERRIFT v0.2.6 - Beastclaw Cherry patch

## Tartalom

- Új Rare skin: `beastclaw_cherry`.
- A skin asset útvonala: `assets/player/skins/beastclaw_cherry/`.
- Beállított animációk:
  - `idle`: 4 frame / irány.
  - `walk`: 6 frame / irány.
  - `attack`: 6 frame / irány.
  - `skill`: 6 frame / irány.
- Új melee basic attack rendszer Beastclaw Cherryhez.
- Új aktív skill: `Savage Rend`.
- A főmenü PC-n háromoszlopos referencia-szerű layoutot kapott.
- Mobilon egyoszlopos, görgethető, safe-area kompatibilis layout van.

## Telepítés

Másold be a ZIP tartalmát a repo gyökerébe, vagy használd a feltöltött GitHub változtatásokat.

A package nem tartalmazza újra az asset mappát, mert a Beastclaw PNG-k már a repóban vannak.
