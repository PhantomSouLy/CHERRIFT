# CHERRIFT v0.7.0 telepítés / Installation

## Magyar

1. Készíts biztonsági mentést a repositoryról.
2. A ZIP **tartalmát** másold közvetlenül a CHERRIFT repository gyökerébe.
3. Engedélyezd a `src/main.js` felülírását.
4. A következő új fájloknak kell megjelenniük:
   - `src/cherrift_v070.js`
   - `v070.css`
5. Futtasd:

```bash
npm install
npm test
```

6. Commit és push után várd meg a GitHub Pages deployt, majd hard refresh.

Ajánlott commitüzenet:

```text
CHERRIFT v0.7.0 Arsenal and enhancement update
```

## English

Extract the ZIP contents directly into the repository root, replace `src/main.js`, run `npm install` and `npm test`, then commit and push.
